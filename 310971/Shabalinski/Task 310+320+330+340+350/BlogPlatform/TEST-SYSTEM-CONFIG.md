# Конфигурация для тестовой системы

## Проблема с SQL запросами

Тестовая система не может подключиться к базе данных `distcomp` для выполнения SQL запросов.

## Требуемые настройки подключения

Тестовая система должна использовать следующие параметры подключения:

- **JDBC URL**: `jdbc:postgresql://localhost:5432/distcomp`
- **Username**: `postgres`
- **Password**: `1902`
- **Schema**: `distcomp`

## Проверка подключения

Убедитесь, что:

1. База данных `distcomp` создана
2. Схема `distcomp` существует в базе `distcomp`
3. Таблицы созданы (с префиксом `tbl_`)
4. Пользователь `postgres` имеет доступ к базе `distcomp`

## Создание базы данных

Если база данных не создана, выполните:

```sql
CREATE DATABASE distcomp;
```

Затем перезапустите приложение, чтобы Liquibase создал схему и таблицы.

## Примечание

REST API работает корректно - все HTTP запросы проходят успешно. 
Проблема только с прямыми SQL запросами тестовой системы к базе данных.
