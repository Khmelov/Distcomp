# Интеграция Spring Security в Blog Publisher API

## Обзор

Добавлена система аутентификации и авторизации с поддержкой двух версий API:
- **v1.0** - Незащищенная версия (обратная совместимость)
- **v2.0** - Защищенная версия с JWT и ролевым доступом

## Конфигурация безопасности

### Технологии
- Spring Security 6
- JWT (JSON Web Tokens)
- BCrypt для хеширования паролей
- Ролевая модель доступа (ADMIN, CUSTOMER)

### Порт и префиксы
- Приложение работает на порту: **24110**
- Незащищенные endpoints: `/api/v1.0/**`
- Защищенные endpoints: `/api/v2.0/**`
